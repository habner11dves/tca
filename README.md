# Nome do projeto
RPG-Ultimate Showdown

## Descrição do projeto
O nosso projeto é sobre um jogo de RPG, que é dividido em 2 modos: modo história e combate 1v1.
Falando um pouco sobre o modo história, ele funcionará assim: temos um reino chamado Eldrath, que foi dividido em 4 reinos diferentes por conta de um dragão do mal, e agora guerreiros corajosos terão que atravessar cada um desses 4 reinos para restaurar a harmonia do reino de Eldrath. Nessa trajetória, você encontrará alguns desafios.
O modo combate funciona assim: o usuário terá 4 opções de guerreiros para escolher para lutar, cada um com uma habilidade diferente, e será um combate do usuário contra a CPU.
